# Surya Pratap Singh Portfolio

Open `index.html` in a browser to view the portfolio.

## Edit Content

- Quick edits: click the `Edit` button on the page. These changes are stored in your browser.
- Full edits: update `portfolio-data.js` to change projects, services, skills, testimonials, links, and all default text.
- Styling: update `styles.css`.
- Resume: `Surya-Pratap-Singh-Resume.pdf` is linked from the hero and social links.
- Profile photo: LinkedIn blocked direct access to the public photo URL during setup. Replace `profileImage` in `portfolio-data.js` with a local image path or public image URL when you have the photo file.

## Files

- `index.html` - page structure
- `styles.css` - responsive layout and visual design
- `script.js` - rendering and edit-mode behavior
- `portfolio-data.js` - editable portfolio content
